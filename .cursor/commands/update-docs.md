# Actualizar documentación — escenarios-politicos

Usar con **`/update-docs`**.

## Archivos

- `README.md` (fuente) ↔ `README.EN.md` (espejo)
- `docs/metodologia/`, `docs/colombia/`, `plans/`

## Qué hacer

1. Contrastar docs con la estructura real del repo.
2. Sincronizar cambios de `README.md` hacia `README.EN.md` (mismo contenido, inglés).
3. Corregir enlaces rotos y rutas de comandos Python.
4. No inventar features que no existan.
5. No commitear salvo petición explícita.

Respuesta al usuario en **español**.
